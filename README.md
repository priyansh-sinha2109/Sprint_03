# 🌦️ Weather-Cast

A responsive weather application built using **HTML, CSS, and Vanilla JavaScript** that fetches real-time weather information using the **OpenWeather API**.

This project was developed as part of **Sprint 3**, focusing on API integration, asynchronous JavaScript, DOM manipulation, geolocation, local storage caching, and dynamic UI updates.

---

## 🚀 Features

* 🌍 Search weather by city name
* 📍 Automatic weather detection using Geolocation API
* 🌡️ Display current temperature
* ☁️ Display weather condition
* 💧 Display humidity
* 💨 Display wind speed
* 🖼️ Dynamic weather icons from OpenWeather
* 🌈 Dynamic background based on weather conditions
* ❄️ Rain & Snow particle animations
* ⚠️ Graceful 404 error handling for invalid cities
* 💾 LocalStorage caching (10-minute cache)
* 📱 Responsive user interface

## ⚙️ How It Works

1. Click **Get Started**.
2. The application requests your current location using the Geolocation API.
3. If permission is granted, your current weather is displayed automatically.
4. If permission is denied, the application loads Delhi weather as a fallback.
5. You can search for any city using the search bar.
6. Weather data is cached in LocalStorage for 10 minutes to reduce unnecessary API requests.
7. The background, animations, and weather icon update automatically according to the current weather.
